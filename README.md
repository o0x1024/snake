
### 项目概述
snake是一个跨平台桌面应用程序，主要用于安全测试和渗透测试工作流程。

### 核心功能

#### 🎯 主要能力
- **会话管理**: 组织和管理多个安全测试会话
- **命令执行**: 执行授权的安全工具和命令
- **文件管理**: 在测试环境中浏览和管理文件
- **插件系统**: 可扩展的架构支持自定义安全工具 todo
- **WebShell生成器**: 用于模板生成工作流程的UI占位符
- **多语言支持**: 支持中英文界面切换

### 安全声明
⚠️ **重要**: 本工具仅设计用于授权的安全测试。
用户必须：
- 对所有测试活动拥有明确授权
- 仅在受控环境中使用
- 遵循负责任的披露实践
